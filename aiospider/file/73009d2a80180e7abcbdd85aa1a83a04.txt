    Find commercial software suppliers for Charities Online
    
        HTML
    


  



  


        
          Details


          

    
            
Internet filing enabled software and online forms are available from HMRC and commercial software suppliers. These can be used to file charity repayment claims online.

HMRC accepts repayment claims filed using any of the products listed in this document.

Get technical support if you’re a charities software developer.
