1 MySQL

1.1 SQLAlchemy

非常方便

1.1.1 sqlacodegen

从数据库的源生成ORM脚本代码文件

xxxxxxxxxx
sqlacodegen --table <tablename> "mysql://<username>:<password>@<host>:<port>/<tablename>" > models.py

1.2 MySQL 8

尽量用低级的版本（数据类型不是升级的足够理由，兼容性最重要）。这次用8遇到的问题：

1.2.1 error 2059

MySQL 8.0报错：error 2059: Authentication plugin 'caching_sha2_password' cannot be loaded

解决方案：https://blog.csdn.net/vkingnew/article/details/80105323
